# 基金从业资格考试 · 复习助手

基金从业资格考试备考工具，提供闪卡、刷题、知识点浏览、模拟考试、错题本等多种复习模式。

## 功能

### 闪卡
翻转式记忆卡片，覆盖知识点和题目两种类型。支持按重要性（必考/高频/常考）、数字考点、易混淆等条件筛选。

### 刷题
随机抽取题目作答，实时反馈正确与否。自动记录错题，支持只做错题或未做题。

### 知识点
按章节浏览所有知识点，支持按重要性、类型（数字考点/易混淆）筛选，可折叠展开详细内容。

### 模拟考试
120 分钟限时闭卷考试，还原真实考试体验。包含答题卡导航、交卷自动评分、错题回顾。

### 错题本
错题自动收集，连对 2 次自动移出错题本。支持按来源（刷题/模考）筛选。

### 进度统计
查看各科目的刷题进度、正确率、闪卡掌握情况。

## 科目

- 基金法律法规与职业道德
- 私募股权投资基金基础知识
- 证券投资基金基础知识

## 使用

直接在浏览器打开 `index.html` 即可使用（建议通过本地服务器运行以避免 CORS 问题）。

```bash
# 方式一：Python 服务器
python3 -m http.server 8080

# 方式二：Node.js 服务器
npx serve .
```

访问 http://localhost:8080

## 部署到 GitHub Pages

1. 进入仓库 Settings → Pages
2. Source 选择 `Deploy from a branch`，Branch 选择 `main`，文件夹选 `/ (root)`
3. 保存后访问 `https://你的用户名.github.io/fund_exam/`

## 数据

- `questions.json` - 题目数据
- `knowledge.json` - 知识点数据
- `mock_exams.json` - 模拟试卷数据

所有数据存储在本地 localStorage，不上传到服务器。
